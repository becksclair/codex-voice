## Findings considered and rejected (do not re-audit)

- **Web dictation / PWA transcription surface** — operator decision (2026-07-07): the web app will NOT do dictation. The TTS-only asymmetry is intentional; do not re-propose.

- **Unauthenticated `/web/config` serving provider API keys; unauthenticated `/web/speech*`; CORS `AllowOrigin::mirror_request()`** — by design. Operator decision (2026-07-07): the service is private, single-user, reachable only over a personal tailnet or localhost; the PWA depends on receiving keys client-side for browser-direct synthesis. Documented in README by plan 015 instead of "fixed".
- **Bearer-token comparison length short-circuit** (`constant_time_eq`) — token is fixed-length 256-bit random; length is public. Not worth changing.
- **Weak bearer token / Tailscale bind posture** — stated design (root AGENTS.md "Deployment Context").
- **Platform adapters (`platform/src/{linux,macos,windows}.rs`) as "duplicated code"** — they are per-target impls of shared core traits (`core/src/platform.rs:24-61`); correct abstraction, no consolidation.
- **Single-implementation core traits** (`AudioRecorder`, `TranscriptionClient`, `SpeechClient`) — each has a real impl + test fake; the trait is a DI seam, not premature abstraction.
- **crossbeam-channel + tokio::sync coexistence** — sync channel is required in the real-time cpal callback; the split is correct.
- **Transcriber `client.rs` vs `upload.rs`** — outbound client vs inbound multipart parsing; distinct responsibilities, no dedup.
- **Lockfile versions above manifest pins** (tokio 1.52 vs pin 1.44 etc.) — normal semver-caret resolution, not drift.
- **thiserror 1+2, bitflags 1+2, winnow ×3, toml_edit ×3 transitive duplicates** — pinned inside GTK3/symphonia/zbus dependency trees; not locally fixable. Watch upstream.
- **GTK3 (gtk-rs 0.18) on the Linux tray** — ~~no local action until upstream moves~~ superseded 2026-07-08: plan 023 migrates the Linux tray to ksni (SNI over D-Bus) and the GTK windows to iced, removing GTK3 entirely.
- **`pcm_to_wav` trailing-odd-byte drop; audio-callback chunk drops on full channel; web-speech-job pruning only on create/status** — reviewed; harmless/intentional/bounded respectively.
- **Streaming realtime transcription** — explicitly ROADMAP "Deferred (Out of Scope)"; not unbuilt intent.
